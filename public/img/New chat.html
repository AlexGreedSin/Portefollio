Créer une procédure qui affiche les noms et les prénoms des clients dont le nom commence par ‘Ca’ en utilisant un cursor qui permet d’extraire les lignes ligne par ligne SQL

Voici un exemple de procédure SQL utilisant un curseur pour extraire les noms et prénoms des clients dont le nom commence par 'Ca':

sql
CREATE PROCEDURE AfficherClientsCa
AS
BEGIN
    DECLARE @NomClient NVARCHAR(50)
    DECLARE @PrenomClient NVARCHAR(50)

    DECLARE ClientCursor CURSOR FOR
    SELECT NomClient, PrenomClient
    FROM Clients
    WHERE NomClient LIKE 'Ca%'

    OPEN ClientCursor
    FETCH NEXT FROM ClientCursor INTO @NomClient, @PrenomClient

    WHILE @@FETCH_STATUS = 0
    BEGIN
        PRINT @NomClient + ' ' + @PrenomClient
        FETCH NEXT FROM ClientCursor INTO @NomClient, @PrenomClient
    END

    CLOSE ClientCursor
    DEALLOCATE ClientCursor
END

Dans cet exemple, nous déclarons un curseur appelé ClientCursor qui sélectionne les noms et prénoms des clients dans la table Clients où le nom commence par 'Ca'. Nous ouvrons ensuite le curseur et utilisons une boucle WHILE pour parcourir chaque ligne extraite par le curseur. À chaque itération de la boucle, nous affichons le nom complet du client en utilisant la commande PRINT. Enfin, nous fermons et libérons le curseur à la fin de la procédure.

Créer une procédure stockée qui compte le nombre de commandes d’un client et affiche le Résultat SQL

Voici un exemple de procédure stockée SQL qui compte le nombre de commandes d'un client spécifique et affiche le résultat:

sql
CREATE PROCEDURE CompterCommandesClient
    @NomClient NVARCHAR(50)
AS
BEGIN
    DECLARE @NbCommandes INT

    SELECT @NbCommandes = COUNT(*)
    FROM Commandes
    WHERE NomClient = @NomClient

    PRINT 'Le nombre de commandes pour le client ' + @NomClient + ' est de ' + CONVERT(NVARCHAR(10), @NbCommandes)
END

Dans cet exemple, nous avons créé une procédure stockée appelée CompterCommandesClient qui prend en paramètre le nom du client dont nous voulons compter le nombre de commandes. Nous déclarons une variable @NbCommandes pour stocker le résultat de notre requête de comptage. Nous utilisons ensuite une requête SELECT pour compter le nombre de commandes correspondant au nom du client spécifié et stockons le résultat dans la variable @NbCommandes. Enfin, nous affichons le résultat de la requête en utilisant la commande PRINT.

Pour exécuter cette procédure stockée, vous pouvez utiliser la commande EXEC en spécifiant le nom de la procédure et le nom du client dont vous voulez compter les commandes. Voici un exemple de code pour exécuter la procédure CompterCommandesClient pour le client "Dupont":

sql
EXEC CompterCommandesClient 'Dupont';

Lorsque vous exécutez ce code, la procédure sera appelée et le nombre de commandes pour le client "Dupont" sera affiché dans la fenêtre de résultats.
